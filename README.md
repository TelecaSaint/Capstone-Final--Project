# 🎮 MathQuest – AI-Powered Learning Platform

MathQuest is an interactive web-based learning platform designed to help students improve their math skills through gamification, real-time feedback, and progress tracking.

---

## 🚀 Features

### 👨‍🎓 Student Features
- Secure login system
- Interactive problem-solving
- XP and level progression
- Assignments and challenges
- Leaderboard ranking
- Progress tracking and reports
- Profile and achievements

### 👨‍🏫 Admin / Teacher Features
- Admin login system
- Manage assignments
- Monitor student performance
- View leaderboard and analytics
- Manage users and content

### 💬 Additional Features
- Messaging system
- Notifications and alerts
- Feedback system for answers
- Session handling and authentication

---

## 🛠️ Technologies Used

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (phpMyAdmin via XAMPP)  
- **Version Control:** Git & GitHub
